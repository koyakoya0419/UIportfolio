# 画像差し替えガイド

## 📸 必要な画像ファイル

以下のファイル名で、このフォルダ（`images/`）に作品画像を配置してください。

### 1. calendar-main.jpg
**カレンダー画面 - シフトを色付きドットで可視化**
- 推奨サイズ: 1200 x 900px (4:3)
- 説明: 月間カレンダービューで、シフトがドットで表示されている画面

### 2. register.jpg
**新規登録画面 - 直感的な入力フォーム設計**
- 推奨サイズ: 1200 x 900px (4:3)
- 説明: シフトを新規登録する入力フォーム画面

### 3. settings.jpg
**設定画面 - 柔軟なカスタマイズ機能**
- 推奨サイズ: 1200 x 900px (4:3)
- 説明: アラームや職場の設定を行う画面

### 4. calendar-detail.jpg
**カレンダー詳細 - シフト情報の一覧表示**
- 推奨サイズ: 1200 x 900px (4:3)
- 説明: カレンダー上で特定の日付を選択した時の詳細画面

### 5. shift-detail.jpg
**シフト詳細表示 - 複数職場の管理対応**
- 推奨サイズ: 1200 x 900px (4:3)
- 説明: 複数職場のシフトを色分けして表示している画面

## 🔧 画像を配置した後の手順

1. 上記のファイル名で画像を `images/` フォルダに保存
2. `index.html` を開く
3. 以下の5箇所を検索して置換：

```html
<!-- 変更前 -->
<img src="images/placeholder.svg" alt="カレンダー画面 - シフトを色付きドットで可視化">

<!-- 変更後 -->
<img src="images/calendar-main.jpg" alt="カレンダー画面 - シフトを色付きドットで可視化">
```

**全て置き換える箇所（5箇所）:**
- `images/placeholder.svg` → `images/calendar-main.jpg`（1箇所目）
- `images/placeholder.svg` → `images/register.jpg`（2箇所目）
- `images/placeholder.svg` → `images/settings.jpg`（3箇所目）
- `images/placeholder.svg` → `images/calendar-detail.jpg`（4箇所目）
- `images/placeholder.svg` → `images/shift-detail.jpg`（5箇所目）

## 💡 ヒント

- 画像は **JPG形式** を推奨（ファイルサイズが小さい）
- 解像度は高すぎないようにする（200KB以下が理想）
- 画像編集ソフトで圧縮すると、読み込みが速くなります

## 🖼️ 画像がない場合

もし実際の画像がまだ準備できていない場合は、以下の方法で一時的に対応できます：

1. **Figmaでスクリーンショット撮影**
2. **プロトタイプのキャプチャ**
3. **デザインモックアップ**

現在はプレースホルダー画像が表示されていますので、サイトは正常に動作します。
